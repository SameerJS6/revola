Mobile keyboard interactions can be challenging when the virtual keyboard pushes content upward. Revola inherits Vaul's automatic repositioning system that automatically adjusts the drawer position when the keyboard opens.

<ComponentPreview name="vaul-default-input-demo" />

### No repositioning

Set `repositionInputs={false}` to disable automatic repositioning and use the browser's default keyboard behavior instead.
