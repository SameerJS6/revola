### Roadmap  

Here’s what’s planned for future updates of Native Modal. If you have suggestions, feel free to contribute!  

#### In Progress 
- Port all dialog components from Origin UI.  
- Add Tailwind v4 Support.

#### Planned Features  
- Nested Native Modal Support → Allow opening multiple `native-modal` instances within each other.
- More Advanced Usage Examples → Showcase real-world implementations and best practices.  
- Support for Different JS Frameworks → Extend `native-modal` beyond React to other frameworks like Vue, Svelte, or Solid.js.  

#### Want to Contribute?
Have an idea or feature request? Open an issue on [GitHub](https://github.com/SameerJS6/native-modal/issues) and help shape the future of Native Modal!
