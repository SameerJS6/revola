## Installation

Native Modal is built using:  
- **[Radix UI Dialog](https://www.radix-ui.com/docs/primitives/components/dialog)** for modals on larger screens.  
- **[Vaul](https://vaul.emilkowal.ski/)** for drawers on mobile.  


It doesn’t require shadcn/ui, but it integrates seamlessly with its styling system and supports installation via the shadcn CLI.

### Install via Shadcn CLI 

If you're already using **shadcn** in your project, you can install Native Modal with:  

```package-install
npx shadcn@latest add http://localhost:3000/r/native-modal.json
```

<MarkdownAccordion title="If you haven't installed shadcn yet">
<Steps>
<Step>
#### Create the project

Run the `init` command to create a new Next.js project or to setup an existing one:

```package-install
npx shadcn@latest init
```

Choose between a Next.js project or a Monorepo.
</Step>

<Step>
#### Add Components
You can now start adding components to your project.

```package-install
npx shadcn@latest add button
```

The command above will add the Button component to your project. You can then import it like this:

```tsx
import { Button } from "@/components/ui/button"

export default function Home() {
  return (
    <div>
      <Button>Click me</Button>
    </div>
  )
}

```
</Step>
</Steps>
</MarkdownAccordion>


### Manual Installation (without shadcn)
If you’d rather not use the shadcn CLI, you can install **Native Modal** manually with just a few steps.

<Steps>
<Step>
#### Install dependencies

Install the required packages:

```package-install
npm install @radix-ui/react-dialog vaul tailwindcss-animate
```
</Step>

<Step>
#### Update `tailwind.config.ts`

Add the animation plugin to your Tailwind config:

```ts title="tailwind.config.ts"
import type { Config } from "tailwindcss";
import tailwindAnimate from "tailwindcss-animate";

export default {
// All you existing configs....

  plugins: [tailwindAnimate],
} satisfies Config;
```
</Step>

<Step>
#### Add the `useMediaQuery` hook

This hook is used internally by Native Modal to detect screen size.

```tsx title="use-media-query.tsx"
"use client";

import { useEffect, useState } from "react";

const useMediaQuery = (query: string) => {
  const [matches, setMatches] = useState<boolean | null>(null);

  useEffect(() => {
    const media = window.matchMedia(query);

    if (media.matches !== matches) setMatches(media.matches);

    const listener = () => setMatches(media.matches);

    window.addEventListener("resize", listener);
    return () => window.removeEventListener("resize", listener);
  }, [query, matches]);

  return matches;
};

export default useMediaQuery;

```
</Step>

## Usage

```tsx 
import { 
  NativeModal, 
  NativeModalTrigger, 
  NativeModalContent, 
  NativeModalHeader, 
  NativeModalTitle, 
  NativeModalDescription 
  } from "@/components/ui/native-modal";
```

```tsx
    <NativeModal>
      <NativeModalTrigger>Open Native Modal</NativeModalTrigger>
      <NativeModalContent>
        <div className="bg-background p-4">
          <NativeModalHeader>
            <NativeModalTitle>Welcome</NativeModalTitle>
            <NativeModalDescription>
              This is a responsive modal that adapts between drawer & modal.
            </NativeModalDescription>
          </NativeModalHeader>
        </div>
      </NativeModalContent> 
    </NativeModal> 
```
